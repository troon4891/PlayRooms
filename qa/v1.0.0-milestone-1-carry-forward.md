## Summary

v1.0.0 covered the carry-forward port of all server and client code from the single-repo HAButtPlugIO-PlayRooms project into the new multi-repo PlayRooms structure. Key deliverables included:

- Express/Socket.IO server with room management, auth, and widgets
- React SPA client
- Acceptance gate and first-boot disclaimer screen
- Guest lobby consent screen
- react-i18next integration for all UI strings
- Plugin loader stub
- Shared relay protocol types
